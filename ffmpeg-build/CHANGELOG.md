# 更新日志

## [1.1.0] - 2025-12-22

### 新增 ✨

- **Docker 状态检查**：编译前自动检查 Docker 是否运行
- **Ubuntu 镜像检查**：自动检查并拉取 Ubuntu 16.04 镜像
- **磁盘空间检查**：警告磁盘空间不足（<2GB）
- **编译前确认**：显示所有检查结果，等待用户确认
- **test-checks.bat**：独立的环境检查脚本

### 改进 🔧

- **错误处理**：提供详细的错误信息和解决方案
- **输出验证**：显示每个库文件的大小（MB/KB）
- **用户反馈**：更清晰的进度和状态信息
- **文档更新**：更新 README、START_HERE 和中文说明

### 技术改进 💻

- 启用延迟变量扩展（`setlocal enabledelayedexpansion`）
- 改进字符串处理和错误级别检查
- 更好的条件判断和流程控制
